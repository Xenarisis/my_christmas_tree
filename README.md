# Welcome to My Christmas Tree
***

## Task
The challenge for this project is to create a christmas tree by taking into consideration 2number

## Description
Create a christmas tree by taking into account 2 number that will define the height and the width.
Concretly it will show in the terminal space to do the void and '*' for the leaf and '|' for the trunk.
It will use some functions to create the tree.

## Installation
There was nothing to install for this project

## Usage
the code take one argument that will define the length and width of the tree, so at the start the code will calculate the number of space that it has to print before the stars and it will start 
to print the tree so first it will print the leaves and after the trunk and that section per section.

```
gcc -o my_christmas_tree ./my_christmas_tree.c
./my_christmas_tree number1 number2
```
